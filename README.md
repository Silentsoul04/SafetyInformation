### 1、启动

  - 运行utils中的run.py文件

  - 命令：
    - python3 run.py

### 2、版本

  - python版本：python3.7.6

### 3、框架

  - scrapy

### 4、安装包

  - 所需第三方安装包和安装包版本：requirements.txt

  - 安装命令：pip3 install -r requirements.txt

### 5、数据库

  - 启动命令：mysql -u root -p root（账号:root,密码:root，本地数据库，端口号为3306）

  - 数据库名称：safe_info

  - 表名称：news

### 6、配置文件

  - settings.py
